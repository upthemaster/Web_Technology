Title: State in React

1. State:
State is used to store data that can change over time.

2. useState Hook:
useState is a hook used to manage state in functional components.

Syntax:
const [state, setState] = useState(initialValue);

3. Example:
const [count, setCount] = useState(0);

4. Key Points:
- State is mutable (can change)
- When state changes → component re-renders
- Used for dynamic behavior

5. Difference between Props and State:

Props:
- Passed from parent
- Read-only

State:
- Managed inside component
- Can be changed

6. Example 1: Counter
Button click increases count

7. Example 2: Toggle
Used to show/hide content

8. Real-life Example:
- Like button count
- Show/hide password
- Login status
